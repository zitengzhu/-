**pwd查看当前的工作目录**

**通过ls来验证当前的工作目录，其实是不恰当的。**

**我们可以通过pwd命令，来查看当前所在的工作目录**

**语法:pwd**

**pwd命令，无选项，无参数，直接输入pwd即可**

**示例:**

\[zzt@localhost ~]$ pwd  #查看当前的工作目录

/home/zzt  # 返回当前工作目录为HOME目录

\[zzt@localhost ~]$ cd /  # 更改当前工作目录为根目录

\[zzt@localhost /]$ pwd  # 查看当前的工作目录

/  # 返回当前的工作目录为根目录
